# Security Policy

This repository contains pet artwork, metadata, previews, and installable zip files.

Please do not open a public issue for sensitive reports, credential leaks, or private data exposure. Send a private message to the maintainer instead.

When contributing, do not include:

- API keys, tokens, or passwords.
- Private screenshots.
- Personal data that is not meant to be public.
- Generated files that include private local paths or machine-specific logs.
